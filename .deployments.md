#### 1. Build the Application
```bash
cd apps/postman-like
pnpm install
pnpm run build

# Next.js standalone mode requires copying static and public files
mkdir -p .next/standalone/apps/postman-like/.next
cp -r .next/static .next/standalone/apps/postman-like/.next/static
cp -r public .next/standalone/apps/postman-like/public
```

#### 2. Run with PM2
Since this is a Next.js application, we run it using the standalone server on port 10012:

```bash
pm2 restart ecosystem.config.cjs
pm2 save
```

#### 3. Nginx Configuration
- sudo nano /etc/nginx/sites-available/postman.rise-confirm.jervi.dev
```nginx
server {
    server_name postman.rise-confirm.jervi.dev;

    location / {
        proxy_pass http://localhost:10012;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    listen 80;
}
```

#### 4. Enable & SSL
```bash
# Enable site
sudo ln -s /etc/nginx/sites-available/postman.rise-confirm.jervi.dev /etc/nginx/sites-enabled/
sudo nginx -t && sudo systemctl reload nginx

# Install SSL
sudo certbot --nginx -d postman.rise-confirm.jervi.dev
```
